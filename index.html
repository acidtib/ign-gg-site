IGNGG

Provide your Discord community with the ability to access Game Guides directly, rather than searching on Google.

Add To Server

Features

Guides

Find guides by game with optional keyword for missions and objectives.

Games

Find a games dedicated page.

Articles

Get the latest article from a game or keyword.

Entire Site

Search the entire site.

Commands

Usage Example:

/ign-guides game:Cyberpunk 2077

with mission or objective:

/ign-guides game:Cyberpunk 2077 keyword:Big in Japan

Usage Example:

/ign-games game:Cyberpunk 2077

Usage Example:

/ign-articles search:CD Projekt Red

Usage Example:

/ign-site search:The Witcher season 2 review
